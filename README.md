# About [Zhou-Yi-Xuan](https://github.com/Zhou-Yi-Xuan/)

愚癸卯年仲秋入南山茶园，现已两年有余。遥想当年青岚五月，听竹语潇潇、闻书声琅琅，
仿佛置于尘世之外，浮生如梦好不快活！
但曲终人散、蓦然回首终究与康庄大道背道而驰，转眼盛夏已逝不复往兮。

当年初冬愚发愤读书却闯进空中楼阁，这一去便是一百二十天光阴，
冬去春来已是梦醒十分，期间虽习得不少手艺但仍不满足，也是这人之野心使然罢。

甲辰龙年年初，愚与旧日至交会于中梁山下，忆起旧地落樱，忆起旧友旧事，忆起旧时理想，
无不如钝刀割心，愚遂决意弃旧图新。梅花三月，回归四方书桌，愚满心欢喜，然事与愿违，
蓬莱钟声环绕耳畔，愚心中惶恐不安，来回往复、上上下下便愈发迷茫，犹如离群之马。
愚深知此非长久之计，遂决意再度重回冷酷仙境，却依旧毫无变化。
仿佛地狱西西弗斯推石上山乎，如此往复两年光阴转瞬即逝。

同窗后生至今未闻吾名，此之谓“英雄不问出处”矣。愚现年虚岁十八实乃风华正茂之时，书无所不读，
也并非全无所获，只是毫无出息，虽看似满腹经论实则内心空虚，愚虽酷爱读书却依旧沦落至此。

纵观天下如今遍是同族相争、官商勾结、勾心斗角之事！想必那武陵桃花源也如此罢。
愚旧友曾有云：“隐逸天山下，不见红尘过。”，如今可好，他却入了学堂等待来年春来报！

卷卷无名，却爱读书；天上圆月，地上铜钱；身旁无人，身后骂名；
静水深流，乾坤未定，你我皆是黑马。
